# 🤖 AI FAQ Chatbot

An intelligent FAQ Chatbot built with **Python**, **Streamlit**, and **RapidFuzz** for the **CodeAlpha AI Internship**.

## 🚀 Features

- 🤖 AI-powered FAQ matching
- 🔍 Smart fuzzy search using RapidFuzz
- 💬 Interactive chat interface
- 📂 FAQ database using CSV
- 📊 Confidence score for responses
- 📝 Chat history
- ⚡ Fast and lightweight
- 🎨 Clean Streamlit UI

## 🛠 Technologies Used

- Python 3.14
- Streamlit
- RapidFuzz
- CSV
- JSON
